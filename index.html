Latest Research

GlobalProtect Security Vulnerabilities

August 2024 Black Hat USA

Research into security vulnerabilities in Palo Alto Networks' GlobalProtect VPN client, including privilege escalation and VPN bypass techniques on macOS and Linux endpoints.

Key Findings

  • Multiple privilege escalation vulnerabilities on macOS and Linux
  • VPN bypass techniques that could compromise network security
  • Design flaws in the GlobalProtect client architecture
  • Security implications of privileged agent deployment
